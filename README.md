## Sobre
Projeto de automação de teste utilizando Selenium para validar o fluxo de cadastro no site da Petlove. 
O teste verifica o funcionamento dos campos obrigatórios do formulário,
 preenchimento de dados e comportamento do sistema durante o processo de registro de usuário.

## Teste Automatizado – Cadastro de Ponto de Doação (Petlov) Descrição
Este projeto contém um teste automatizado desenvolvido para validar o fluxo de cadastro de um ponto de doação no sistema Petlov. 
O teste simula a interação de um usuário com o formulário de cadastro e verifica se o sistema realiza o registro corretamente.

#Objetivo do Teste

Validar o funcionamento do formulário de cadastro, verificando o preenchimento dos campos, 
seleção do tipo de pet e confirmação do cadastro.

## Cenário de Teste

Acessar a página de cadastro.

Preencher os campos do formulário.

Buscar o endereço pelo CEP.

Informar número e complemento do endereço.

Selecionar o tipo de pet.

Enviar o formulário.

Resultado esperado: o sistema deve exibir a mensagem informando que o ponto de doação foi cadastrado com sucesso.

## Tecnologias Utilizadas

Java Selenium Selenide JUnit 5 ChromeDriver

## Como executar o projeto

Clonar o repositório.

Abrir o projeto em uma IDE como VS Code.
Executar a classe de teste.
